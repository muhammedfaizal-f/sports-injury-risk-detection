# Datasets

## COCO Keypoints Dataset
Source: cocodataset.org — sample images saved for reference to the 17-keypoint
body format, matching MediaPipe/OpenPose output.

## MPII Human Pose / SportsPose / Human3.6M
Identified as candidate datasets for pose validation in later milestones.
Full access requires registration; not required for M1/M2 scope.

## FIFA Injury Dataset (Reference)
Reference source for injury trend / risk factor modeling in Milestone 3.
