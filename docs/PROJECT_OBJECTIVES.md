# Sports Injury Risk Detection from Video — Project Objectives

## 1. Problem Statement
Build a web platform that analyzes an athlete's movement from video, flags risky
biomechanical patterns, and recommends corrective exercises before an injury occurs.

## 2. Core Workflow
Athlete uploads video → AI detects body joints → movement analyzed → injury risk
predicted → risk score generated → corrective exercises + report shown.

## 3. User Roles
Athlete, Coach, Physiotherapist, Sports Scientist, Administrator.

## 4. Milestone Roadmap
- M1 (Weeks 1-2): Setup, auth, athlete profiles, dataset prep — DONE
- M2 (Weeks 3-4): Pose estimation, biomechanical analysis — IN PROGRESS
- M3 (Weeks 5-6): Injury prediction, risk scoring, recommendations
- M4 (Weeks 7-8): Dashboards, testing, deployment
